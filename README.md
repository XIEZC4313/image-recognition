# AI 图像识别

一个基于浏览器端本地图像分析的识别网站，无需下载模型，即时分析图像特征。

## 功能特性

- **拖拽/点击上传** - 支持 JPG、PNG、GIF、WebP 格式
- **场景标签识别** - 自动识别图像中的植物、天空、人像、亮暗场景等
- **主要颜色提取** - 提取图像中的主色调并显示占比
- **图像特征分析** - 亮度、对比度、饱和度、边缘密度等多维度分析
- **边缘检测可视化** - Sobel 算子实时边缘检测并展示结果
- **暖冷色调分析** - 分析图像的色温倾向

## 图像分析算法

### 颜色量化
将像素颜色空间量化为 32 级，统计主要颜色分布。

### 亮度分析
使用加权公式计算人眼感知亮度：
```
Y = 0.299R + 0.587G + 0.114B
```

### 边缘检测
使用 Sobel 算子进行边缘检测：
- 计算水平梯度 Gx 和垂直梯度 Gy
- 边缘强度 = sqrt(Gx² + Gy²)

### 场景识别
基于颜色分布、亮度、对比度、边缘密度等特征，推断图像内容标签：
- 植物/自然（绿色区域占比）
- 天空/水（蓝色区域占比）
- 人像/肤色（肤色区域占比）
- 亮暗场景、高/低对比度等
